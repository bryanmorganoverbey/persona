# Housing

Preferences related to residential properties. Covers minimum requirements for size, layout, amenities, and other factors considered when evaluating a place to live. Splits time between the US (mid-April to Nov 1) and Mexico (Nov 1 to mid-April).

## Files

| File | Description |
|------|-------------|
| [minimum-requirements.md](./minimum-requirements.md) | Shared baseline requirements that apply to any property regardless of country. |
| [us-requirements.md](./us-requirements.md) | US-specific requirements: ownership target, South US (Tennessee/Florida), quiet residential, unfurnished, in-unit laundry. |
| [mexico-requirements.md](./mexico-requirements.md) | Mexico-specific requirements: renting in Oaxaca, walkable urban, furnished, clothesline OK, no parking. |
| [deal-breakers.md](./deal-breakers.md) | Instant disqualifiers -- if any item is present, walk away from the property. |
| [inspection-checklist.md](./inspection-checklist.md) | Room-by-room walkthrough checklist for use when touring a property in person. |
| [move-in-essentials.md](./move-in-essentials.md) | Items the property must provide and items to bring or buy on day one. |
