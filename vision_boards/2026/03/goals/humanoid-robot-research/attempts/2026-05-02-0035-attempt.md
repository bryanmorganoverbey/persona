## Detailed Project Profiles

### 1. Berkeley Humanoid Lite (Highest Recommendation)

**Repository:** https://github.com/HybridRobotics/Berkeley-Humanoid-Lite

**Overview:**
- Developed by UC Berkeley's Hybrid Robotics Lab
- Sub-$5,000 cost target using 3D-printed components
- Modular design with customizable gearboxes
- Full software stack with Isaac Lab integration
- Actively maintained (v1.1.0 released September 2025)

**Specifications:**
- Height: ~60cm (miniature humanoid)
- DOF: 20+ degrees of freedom
- Actuation: Custom 3D-printed gearboxes with standard motors
- Materials: 3D-printed PLA/ABS/TPU
- Electronics: Widely available components

**Software Stack:**
- Python-based control framework
- Isaac Lab simulation
- ROS2 integration ready
- RL training support (reinforcement learning)
- Docker container support

**Documentation:**
- Complete GitBook documentation
- CAD release files available
- Motor configuration JSON
- Docker support
- Sim2real deployment scripts

**Strengths:**
- Most accessible for beginners
- Comprehensive documentation
- Active community (Discord)
- Academic backing (Berkeley)
- Clear BOM and sourcing guide

---

### 2. Roboto Origin (Most Advanced DIY)

**Repository:** https://github.com/Roboparty/roboto_origin

**Overview:**
- Developed by RoboParty (Beijing startup)
- Full-stack open source released January 2026
- Can actually walk and run (demonstrated)
- Complete hardware + software + training pipeline

**Architecture - 5 Sub-repositories:**

| Module | Repository | Purpose |
|--------|-------------|---------|
| Atom01_hardware | Roboparty/Atom01_hardware | Mechanical structure, CAD, PCB |
| atom01_deploy | Roboparty/atom01_deploy | ROS2 driver, middleware |
| atom01_train | Roboparty/atom01_train | RL training, Sim2Sim |
| atom01_description | Roboparty/atom01_description | URDF models |
| atom01_firmware | Roboparty/atom01_firmware | Embedded software |

**Specifications:**
- Full-size humanoid (~1.2m height)
- Can achieve walking and running
- Actuators: Standard servo motors
- Procurement: Taobao/Jialichuang compatible
- License: GPLv3

**Software Stack:**
- ROS2 Humble
- Isaac Lab training
- MuJoCo simulation support
- Reinforcement learning (RSL-RL)
- USB2CAN communication

**Strengths:**
- Proven locomotion capability
- Complete training pipeline
- Hardware DIY-feasible
- Active development

### 3. pib (Best for Education)

**Website:** https://pib.rocks/

**Overview:**
- German project by isento GmbH
- Designed for 3D printing and education
- Version 4 released November 2025
- Used in 70+ schools/institutions

**Specifications:**
- Height: ~40cm (compact)
- 3D-printable STL files available
- Servo motor based
- Microphone array for voice interaction
- RGB programmable buttons

**Features:**
- Simplified motor calibration (v4 improvement)
- Sound direction detection
- Open software components
- AI integration support

**Community:**
- 1,800+ community members
- Active Discord
- Awards: German Design Award 2025, German Innovation Award 2025

**Cost Estimate:**
- ~$800-1,500 (consumer 3D printing)

**Strengths:**
- Educational focus
- Simplest assembly
- Award-winning design
- School-proven

### 4. Poppy Humanoid

**Repository:** https://github.com/poppy-project/poppy-humanoid

**Overview:**
- Longer-established project (pre-2020)
- French origin
- Modular humanoid design
- Research and education focus

**Specifications:**
- Height: ~80cm
- 25 degrees of freedom
- 3D-printed modular parts
- Pololu or equivalent servos

**Software:**
- Python-based
- ROS compatible
- V-REP simulation support

**Status:**
- More mature but less active development
- Good documentation legacy

## Hardware Component Considerations

### Typical Actuator Options

| Type | Torque Range | Cost | Notes |
|------|--------------|------|-------|
| Hobby servos (SG90, MG996R) | 1-3 kg⋅cm | $5-20 | Education only |
| Standard servos (MG995, HITEC) | 3-10 kg⋅cm | $15-50 | Basic projects |
| High-torque servos (MX-64, LX-824) | 5-30 kg⋅cm | $50-200 | Research grade |
| Custom gearboxes + motors | Variable | $50-300 | Berkeley Lite approach |

### 3D Printing Requirements

| Component | Material | Printer |
|-----------|----------|---------|
| Structural parts | PLA, PETG | Any FDM |
| Gears | PLA, Nylon | Metal hotend preferred |
| Joints | TPU | Flexible filament |
| Enclosures | ABS, PETG | Enclosed recommended |
