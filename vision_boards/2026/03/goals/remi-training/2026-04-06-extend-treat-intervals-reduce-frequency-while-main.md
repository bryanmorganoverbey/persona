# Remi Public Behavior Training: Extending Treat Intervals Protocol

## Executive Summary

This document provides a comprehensive protocol for extending treat intervals during Remi's public behavior training while maintaining reliable calm behavior at cafes, restaurants, and events. The protocol is based on behavioral science principles of reinforcement schedules and includes a practical step-by-step approach, data tracking templates, and troubleshooting guidance.

---

## 1. Understanding Reinforcement Schedules

### Why Extend Treat Intervals?

When initially teaching "get under," continuous reinforcement (rewarding every time) helps Remi quickly learn the behavior. However, to achieve reliable long-term performance in public settings, we need to transition to **intermittent reinforcement**. This is based on B.F. Skinner's research showing that behaviors maintained on variable schedules are more resistant to extinction and more durable in real-world conditions.

### Key Principles

| Schedule Type | Description | Best For |
|---------------|-------------|----------|
| Continuous (CRF) | Reward every time | Initial learning |
| Fixed Interval (FI) | Reward after set time | Building duration |
| Variable Interval (VI) | Random intervals (average time) | Maintaining calm, preventing extinction |
| Variable Ratio (VR) | Random number of behaviors | Perfecting & maintaining |

### For Remi's Training: Variable Interval (VI)

Since the goal is calm, quiet duration under a table, **variable interval scheduling** is ideal. The dog doesn't know exactly when the treat is coming, so she maintains a steady, calm state rather than checking in frequently or getting frustrated.

## 2. Step-by-Step Extension Protocol

### Phase 1: Baseline (Current State)
**Goal:** Establish current treat frequency and behavior reliability

- Current interval: ________ seconds (estimate)
- Success rate: ________% of sessions
- Typical session duration: ________ minutes

### Phase 2: Gradual Extension

**Step 1: Increase interval by 25%**
- If currently treating every 30 seconds, extend to ~37 seconds
- Use a timer or mental count
- Maintain 90%+ success rate before proceeding

**Step 2: Hold at new interval for 3-5 sessions**
- Track each session
- Only proceed if behavior remains reliable

**Step 3: Increase by another 25%**
- Continue gradual progression
- Example progression: 30s → 37s → 45s → 55s → 67s → 80s → 95s → 2min

**Step 4: Vary the intervals**
- Once at a consistent interval, add randomness
- Example (targeting 60s average): 30s → 90s → 45s → 75s → 60s → 90s → 30s → 75s

### Phase 3: Maintenance

- Continue with variable interval schedule indefinitely
- Occasionally reward with higher-value treats for exceptional calm
- Never completely stop rewarding (behaviors can extinguish)

## 4. Practical Implementation Guide

### Session Setup

1. **Pre-session:** Ensure Remi has had adequate exercise (not exhausted, but not hyped)
2. **Location:** Start with familiar, low-distraction cafes
3. **Command sequence:**
   - "Remi, get under"
   - Remi goes under chair/table
   - Wait for calm (sitting or lying quietly)
   - Deliver treat quietly (drop to floor, don't make eye contact)
   - Reset timer

### During Session

- **Watch for calm signals:** Settled posture, relaxed ears, gentle breathing
- **Avoid:** Eye contact that might prompt talking, excessive verbal praise
- **Timing:** Deliver treat BEFORE Remi gets antsy or starts looking around
- **Variable timing:** Randomize intervals to keep Remi guessing (maintains attention)

### Ending Sessions

- **End on success:** Always finish while Remi is still calm and engaged
- **Final reward:** Give a slightly larger/higher-value treat at end
- **Duration:** 20-45 minutes typical, adjust based on Remi's state

## 5. Troubleshooting

### Problem: Remi gets restless when interval extends

**Solutions:**
- Reduce extension increment (try 10-15% instead of 25%)
- Return to previous interval for 2-3 sessions
- Add more high-value treats during longer intervals
- Increase exercise before session

### Problem: Barking or noise returns

**Solutions:**
- Immediately return to continuous reinforcement (reward every calm moment)
- Reduce session length
- Move to quieter location
- Check for triggers (other dogs, waiters, etc.)

### Problem: Remi won't settle under table

**Solutions:**
- Practice "get under" command at home first
- Bring comfort item (familiar blanket or bed)
- Use calming aids (thundershirt, calming spray)
- Start with shorter sessions

### Problem: Behavior differs between familiar and new locations

**Solutions:**
- Keep treats more frequent in new environments
- Practice "get under" in various locations with high reward rate
- Build confidence with neutral locations before high-value destinations

## 6. Advanced Techniques

### Fading Food Rewards (Long-term)

When behavior is extremely reliable (95%+ across multiple environments):

1. **Partial reinforcement:** Reward 3 out of 5 calm episodes
2. **Gradual reduction:** Move to 2 out of 5, then 1 out of 3
3. **Random rewards:** Use variable ratio (like slot machines - keeps behavior strong!)
4. **Social rewards:** Mix in praise, petting, play as rewards
5. **Environmental rewards:** Allow sniffing, exploration as rewards

**Note:** For public access behavior, maintain some food reinforcement. Complete fading not recommended as behavior must remain reliable in distracting environments.

### Adding Duration Beyond Treats

Once intervals are extended, add duration challenges:

- Start timing from "get under" command
- Count full minutes of calm before first treat
- Gradually increase to 10+ minutes of calm before first reward

## 8. Key Takeaways

1. **Go slow:** Remi has learned the behavior; now we're building reliability. Rushing leads to regression.

2. **Track everything:** Data reveals patterns - what's working, what needs adjustment.

3. **Return to baseline when needed:** Not a failure, just part of learning.

4. **Variable is better than fixed:** Random intervals maintain better behavior than predictable ones.

5. **Never fully eliminate treats:** For public access reliability, maintain some reinforcement indefinitely.

6. **End on success:** Every session should end with Remi in a calm state, even if that means more frequent treats at the end.
