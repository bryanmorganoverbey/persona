## Key Metrics to Track

1. **Duration** - How long Remi remains calm under the table/chair
2. **Treat Interval** - How frequently treats are needed to maintain behavior
3. **Incident Count** - Number of barking/whining incidents per session
4. **Recovery Time** - How quickly Remi recovers after a distraction
5. **Environment Complexity** - New vs. familiar locations

## Usage Instructions

1. **Start a session** - Note start time when Remi goes under
2. **Track continuously** - Note any disruptions, treats given, and time
3. **End session** - Record total calm duration (minus any incidents)
4. **Rate behavior** - Use 1-5 scale for quick reference
5. **Weekly review** - Calculate averages and adjust treat intervals
6. **Monthly progress** - Document growth and plan next environment tests
