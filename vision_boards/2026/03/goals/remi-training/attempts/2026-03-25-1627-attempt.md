### Before Session
- [ ] Bring high-value treats (cut into small pieces)
- [ ] Bring water bowl
- [ ] Know the target cafe and seating spot
- [ ] Set realistic session duration goal (start with 20-30 min)

### During Session
1. Arrive, order, get seated
2. Give "get under" command — Remi goes under table
3. Initial treat delivered immediately upon compliance
4. Deliver treats at interval (see schedule below)
5. If Remi becomes restless: wait for calm before treating
6. Ignore minor movements; reward stillness
7. If barking/disruption: brief timeout (ignore), then retry

- [ ] Log session data (see Section 7)

## 5. Treat Interval Extension Protocol

Based on research, use a **variable interval reinforcement schedule** — this is most effective for maintaining calm behavior without creating treat dependency.

### Phase 1: Initial Training (familiar cafes)
- **Interval:** Every 30-60 seconds
- **Rationale:** High frequency reinforces that calm = rewards

### Phase 2: Building Reliability
- **Interval:** Every 60-90 seconds
- **Method:** Randomize slightly (not perfectly timed)
- **Rationale:** Variable schedule strengthens behavior persistence

### Phase 3: Extending Duration
- **Interval:** Every 90-120 seconds
- **Goal:** Increase duration while maintaining calm

### Phase 4: Maintenance
- **Interval:** Every 2-5 minutes (variable)
- **Rationale:** Dog learns that calm behavior is always rewarded, but timing is unpredictable

**Interval Extension Guidelines:**
- Only advance to next interval phase when 90% success rate achieved
- If regression occurs, return to previous interval
- Always use high-value treats during extension phase
- Never go more than 5 minutes without acknowledgment (touch/voice counts)

## 6. Gradual Desensitization to New Environments

When ready to test in **new/unfamiliar restaurants** (March target), follow this protocol:

### Step 1: Pre-Visit
- [ ] Research layout — is there space under tables?
- [ ] Visit during off-peak hours (fewer stimuli)
- [ ] Bring familiar items (blanket, favorite toy)

### Step 2: Initial New Environment Visit
- [ ] Keep first visit short (15 minutes max)
- [ ] Increase treat frequency initially (every 30 sec)
- [ ] Use higher value treats than familiar cafes
- [ ] If Remi is stressed, end session early — no penalty

### Step 3: Build to Consistency
- [ ] Return to same new location 2-3 times before adding another
- [ ] Gradually extend duration
- [ ] Normalize treat schedule to match familiar cafes

### Step 4: Add Distractions
- [ ] Progressively add: more people, background noise, other dogs at distance
- [ ] Reward calm during mild distractions
- [ ] Increase distance if threshold exceeded

## 8. Troubleshooting

| Issue | Likely Cause | Solution |
|-------|--------------|----------|
| Won't go under table | Not enough space / uncomfortable | Choose different table; use mat |
| Barking at servers | Over-stimulation | Increase distance; higher treat freq |
| Treat-seeking/whining | Too hungry / too frequent treats | Reduce treats slightly; ensure fed before |
| Lying down but restless | Needs more exercise before | Walk before cafe visit |
| Regression | Stress / environment change | Return to shorter intervals, familiar location |

---

## 9. Key Takeaways from Research

1. **Variable interval reinforcement** is most effective for maintaining calm behavior long-term (not fixed intervals)
2. **Gradual desensitization** — never rush new environments; start with short, positive sessions
3. **Consistency is critical** — same command, same table, same expectations each visit
4. **Track everything** — data reveals patterns and progress that memory misses
5. **Set up for success** — ensure Remi is exercised, not too hungry or too full before training
