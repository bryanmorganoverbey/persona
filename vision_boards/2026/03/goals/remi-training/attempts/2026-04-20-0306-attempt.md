## 1. Session Tracking Template

Use this format for each training session:

### Session Log Entry

| Field | Description |
|-------|-------------|
| **Session ID** | Auto-increment: 001, 002, 003... |
| **Date** | YYYY-MM-DD |
| **Start Time** | HH:MM |
| **Location Name** | Name of café/restaurant/venue |
| **Location Type** | Familiar / New / Unfamiliar |
| **Duration Under Chair (minutes)** | Total calm time in minutes |
| **Treats Given** | Count of treats during session |
| **Treat Interval (minutes)** | Average minutes between treats |
| **Challenges Faced** | Description of any disturbances |
| **Distractions Handled** | Yes/No — Did Remi stay calm? |
| **Behavior Rating** | Excellent / Good / Needs Work |
| **Notes** | Any observations |

## 5. Progress Metrics

### Key Performance Indicators

| Metric | Target | How to Measure |
|--------|--------|----------------|
| **Average session duration** | 30+ min by end of March | Sum of duration / number of sessions |
| **Treat interval** | Extend from 2 min → 5+ min | Track average interval between treats |
| **New location success** | Calm in 3+ new locations/month | Count new location sessions with "Good" or better |
| **Distraction handling** | 90%+ success rate | Sessions with distractions handled / total sessions |

---

## 6. Analysis Guidance

### Interpreting Data

- **Duration increasing** = Behavior becoming more reliable
- **Treat interval increasing** = Reinforcement fading working
- **Behavior rating improving** = Progressing from "Needs Work" → "Good" → "Excellent"
- **New locations rated "Good"+** = Ready for next challenge level

### Progress Milestones

- [ ] **Stage 1:** 15+ min calm in familiar locations
- [ ] **Stage 2:** 30+ min calm in familiar locations
- [ ] **Stage 3:** 20+ min calm in new locations
- [ ] **Stage 4:** Extended intervals (5+ min between treats)
- [ ] **Stage 5:** Reliable in all café/restaurant environments
