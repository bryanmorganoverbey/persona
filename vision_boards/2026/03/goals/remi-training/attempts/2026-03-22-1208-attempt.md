## 3. Treat Interval Extension Protocol

Based on reinforcement learning principles:

### Phase 1: High-Rate Reinforcement (Foundation)
- **Interval:** Every 5-10 seconds
- **When:** New location or early training
- **Purpose:** Build association: "get under" = rewards

### Phase 2: Fixed Interval Schedule
- **Interval:** Every 30 seconds (gradual increase)
- **When:** Behavior is reliable (80%+ calm sessions)
- **Method:** Reward after interval passes IF Remi is calm

### Phase 3: Variable Interval Schedule
- **Interval:** Random 45-120 seconds
- **When:** Solid reliability (90%+ calm sessions)
- **Method:** Unpredictable timing maintains attention
- **Why:** Prevents "checking out" behavior; keeps Remi alert

### Key Principles (from AKC & Karen Pryor Clicker Training):
1. **Never reward during excitement** — wait for calm to return
2. **Randomize intervals** once behavior is strong
3. **Reset on breaks** — if Remi gets up, restart interval
4. **Quality over quantity** — fewer, higher-value treats

## 5. Training Tips for Consistent Calm Behavior

### Pre-Session Checklist
- [ ] Feed Remi before leaving (reduces food motivation)
- [ ] Exercise briefly before (release excess energy)
- [ ] Bring high-value treats (chicken, cheese, etc.)
- [ ] Choose corner/quiet table when possible

### During Session
- Use calm, quiet voice
- Reward ONLY when Remi is calm — never during barking
- If Remi barks, wait for silence BEFORE treating
- Practice "get under" command before expecting calm
- Keep sessions under 30 minutes initially

### Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Restlessness | Shorter intervals, more movement breaks |
| Barking at others | Use body to block, redirect with treat |
| Wanting to leave | End session on positive note |
| Begging for food | Feed full meal before session |
