### Remi Calm Behavior Duration Tracker

A complete tracking system has been created with the following components:

#### 1. Session Tracking Template

| Field | Description |
|-------|-------------|
| **Date** | Date of training session |
| **Location** | Cafe/Restaurant name (familiar vs. new) |
| **Start Time** | When session began |
| **End Time** | When session ended |
| **Total Session Duration** | Total time at location |
| **"Get Under" Start** | Time Remi went under chair/table |
| **Calm Behavior Start** | When calm behavior began |
| **Calm Behavior End** | When calm ended (any noise/movement) |
| **Total Calm Duration** | Duration of continuous calm |
| **Treat Interval** | Time between treats (extended as trained) |
| **Incidents** | Any barking/whining/噪声 |
| **Environment Notes** | Noise level, crowds, etc. |
| **Success Rating** | 1-5 scale |

#### 2. Key Metrics to Track

**Primary Metric: Duration**
- How long Remi maintains calm, quiet behavior under the table/chair
- Track in minutes and seconds
- Measure from when calm begins to any disruption

**Secondary Metrics:**
- **Frequency**: Number of times "get under" command was given
- **Treat Interval**: How often treats were needed to maintain calm
- **Incidents**: Any barking, whining, or disruptive behavior

#### 3. Duration Progression Guide

| Training Phase | Target Calm Duration | Treat Interval |
|----------------|---------------------|----------------|
| Week 1-2 | 5-10 minutes | Every 1-2 min |
| Week 3-4 | 10-15 minutes | Every 2-3 min |
| Week 5-6 | 15-20 minutes | Every 3-5 min |
| Week 7+ | 20-30+ minutes | Every 5+ min |

#### 6. Instructions for Use

**During Each Session:**
1. Record start time when arriving at location
2. Give "get under" command
3. Start timer when Remi settles into calm position
4. Treat at regular intervals per training phase
5. Note any disruptions (time, trigger, duration)
6. Record end time when leaving

**After Each Session:**
1. Calculate total calm duration
2. Note treat interval used
3. Rate success (1-5)
4. Add any environmental notes

**Weekly Review:**
1. Calculate average calm duration
2. Note longest session
3. Determine if ready to extend treat intervals
4. Plan next week's locations
