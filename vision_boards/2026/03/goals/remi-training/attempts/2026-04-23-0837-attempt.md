### 🎯 Key Metrics & Definitions

| Metric | How to Measure | Target |
|--------|----------------|--------|
| **Calm Duration** | Time from "get under" command to release, minus any disruptions | 15-30 min initially, extend to 45-60 min |
| **Calm Percentage** | (Calm time ÷ total session time) × 100 | ≥90% |
| **Treat Interval Extension** | Time between treats while maintaining calm behavior | Progress from 2-3 min → 5-10 min → 15+ min |
| **Incident Rate** | Number of disruptions per session | Decreasing trend (ideally 0) |
| **Environmental Adaptability** | Performance in new vs. familiar locations | Consistent across both |

**Success Criteria:**
- ✅ Calm behavior ≥20 minutes in familiar environments
- ✅ Calm behavior ≥15 minutes in new environments
- ✅ Treat interval extended by 50% within 2 weeks
- ✅ Zero incidents in 3+ consecutive sessions
- ✅ Calm percentage ≥90% consistently

### 📝 Quick Reference: Session Recording Checklist

**Before Session:**
- [ ] Note location and environment type
- [ ] Start timer when Remi settles under table/chair

**During Session:**
- [ ] Record time of each treat given
- [ ] Note any incidents with timestamp
- [ ] Track when Remi becomes restless

**After Session:**
- [ ] Calculate total calm duration
- [ ] Determine max treat interval achieved
- [ ] Rate overall session (1-5 stars)
- [ ] Add entry to persistent data table

---

### 🔬 Data Analysis Guidelines

**Progress Indicators to Watch:**
1. **Increasing calm duration** → Remi can hold calm state longer
2. **Increasing max treat interval** → Reward frequency can decrease while maintaining behavior
3. **Decreasing incident rate** → Behavior becoming more reliable
4. **Similar performance in new vs. familiar** → Generalization improving
5. **Higher calm percentage** → More consistent reliability

**Trends to Track Weekly:**
- Average calm duration per session
- Average treat interval progression
- Number of incident-free sessions
- Performance differential (new vs. familiar environments)
