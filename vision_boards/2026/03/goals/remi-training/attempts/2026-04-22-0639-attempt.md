## 4. Data Collection Guidelines

### How to Time Sessions

1. **Start Timer**: When Remi is given "get under" and settles
2. **Stop Timer**: When Remi gets up, barks, or is asked to come out
3. **Record Actual Calm Time**: Net minutes of confirmed calm behavior (exclude disruptions)

### Rating Scale (Competency 1-10)

| Rating | Description |
|--------|-------------|
| **10** | Perfect — immediate settle, no disruptions, full calm duration |
| **8-9** | Excellent — minor adjustment needed, one brief alert |
| **6-7** | Good — some reinforcement needed, recovered quickly |
| **4-5** | Moderate — multiple reinforcements, some vocalization |
| **2-3** | Struggling — difficult to settle, significant disruptions |
| **1** | Need to retry — could not complete session |

### Environment Classification

- **Familiar**: Previously visited venue where Remi has trained before
- **New**: First-time venue to test generalization

## 7. Progress Analysis Framework

### Key Questions to Answer Monthly:

1. **Is calm duration increasing?** → Track average calm minutes per session
2. **Are treat intervals extending?** → Track average treat interval over time
3. **Is competency improving?** → Track average rating
4. **Are new environments handled as well as familiar?** → Compare F vs N ratings
5. **Are incidents decreasing?** → Track incident frequency

### Milestone Targets (Adjust Based on Goals):

| Milestone | Calm Duration | Treat Interval | Environment |
|-----------|---------------|-----------------|--------------|
| **Phase 1** | 15 min | 3-5 min | Familiar only |
| **Phase 2** | 30 min | 5-10 min | Familiar + 1 new |
| **Phase 3** | 45 min | 10-15 min | Mix of environments |
| **Phase 4** | 60+ min | 15-20 min | New environments reliably |
| **Mastered** | 90+ min | 20-30 min | Any public setting |
