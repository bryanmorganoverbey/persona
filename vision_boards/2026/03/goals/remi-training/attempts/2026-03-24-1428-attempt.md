## 3. GUIDELINES FOR ACHIEVING CONSISTENT CALM BEHAVIOR

### Phase 1: Build Reliability in Familiar Cafes

**Environment Selection:**
- Start with 2-3 regular cafe locations
- Choose quieter times (early morning, mid-afternoon)
- Consistent seating location when possible
- Short sessions initially (15-20 minutes)

**Training Protocol:**
1. **Entry**: Give "get under" command immediately upon sitting
2. **Initial Reward**: Treat for immediate compliance
3. **Sustained Rewards**: Treat every 30-60 seconds for first 3-5 sessions
4. **Gradual Extension**: Increase intervals using variable schedule

### Phase 2: Extend Treat Intervals (Variable Interval Schedule)

**Why Variable Intervals?**
According to the American Kennel Club, variable interval reinforcement creates steady, reliable behavior without the dog expecting a treat every time. This builds resilience and prevents "paycheck-only" behavior where the dog only works when treats are visible.

**Interval Progression:**
| Stage | Interval Range | Example Timing |
|-------|---------------|----------------|
| Early | 30-60 seconds | 30s, 45s, 60s, 35s, 50s |
| Mid | 60-90 seconds | 70s, 85s, 60s, 90s, 75s |
| Advanced | 90-120 seconds | Variable between 90s-120s |
| Maintenance | Random | Unpredictable timing |

**Key Principle**: Always keep Remi guessing about when the next treat comes, but never let the interval get so long that calm behavior breaks down.

### Phase 3: Reduce Treat Frequency While Maintaining Behavior

**Signs Remi is Ready for Fewer Treats:**
- Settles immediately under table without prompting
- Stays calm during brief distractions (server passing, other customers)
- Doesn't bark or whine when treat doesn't come immediately
- Maintains relaxed body language (lying down, eyes soft)

**Transition Strategy:**
- Replace every 3rd treat with praise/petting
- Use "life rewards" (opportunity to sniff, water break) intermittently
- Mix high-value and low-value treats
- Gradually phase to verbal praise only for successful intervals

## 5. TROUBLESHOOTING GUIDE

### Common Issues & Solutions

| Issue | Likely Cause | Solution |
|-------|-------------|----------|
| Won't settle under table | Too stimulated | Practice "place" command at home first |
| Barks at servers | Anticipation | Treat BEFORE server approaches |
| Gets up repeatedly | Boredom/need to move | Shorter sessions, more frequent rewards initially |
| Only calm with visible treats | Dependent on continuous reinforcement | Transition to variable interval ASAP |
| Different behavior at different cafes | Environmental association | Build familiarity through repeated visits |

### For New/Familiar Cafe Differences:
- **Familiar**: Higher expectations, longer intervals acceptable
- **New**: Start fresh with shorter intervals, rebuild confidence
