### Recommended Data Collection Approach

1. **Use a phone timer** - Start when "get under" is given, stop when Remi is released or becomes agitated

2. **Record "calm duration" specifically** - Only count time when Remi is relaxed (lying down, resting, not alert/standing)

3. **Note disturbances** - Record what triggered any disruption (person, dog, noise, etc.)

4. **Track treat intervals** - Note the seconds between treats to measure progression toward extended intervals

5. **Rate each session** - 1 = failed (barking/agitation throughout), 5 = perfect (full session calm)

---

### Key Metrics to Track

| Metric | Description | Target |
|--------|-------------|--------|
| Calm Duration | Minutes Remi remained relaxed under table | Increase over time |
| Disturbance Recovery | Time to return to calm after disturbance | < 2 minutes |
| Treat Interval | Seconds between treats | Gradually increase |
| Session Success Rate | % of sessions with no major disruptions | > 80% |
