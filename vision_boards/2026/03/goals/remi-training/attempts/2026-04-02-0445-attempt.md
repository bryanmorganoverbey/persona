# Extending Treat Intervals: Remi's Public Behavior Training Guide

## Understanding the Science

### Why Variable Interval Schedules Work Best

For calm behavior maintenance (like Remi's "get under"), **variable interval (VI) schedules** are optimal because:

1. **Steady response rate** - Dogs maintain consistent behavior without anticipating exactly when treats come
2. **Resistant to extinction** - Behavior persists even when rewards become unpredictable
3. **Builds duration** - Teaches Remi to hold calm behavior for progressively longer periods
4. **Prevents "paycheck mentality"** - Unlike fixed schedules, Remi won't only behave when she sees a treat

### Key Principle: Start Slow, Progress Gradually

Research indicates that extending intervals too quickly can cause:
- Frustration and stress
- Attention-seeking behaviors (barking, pawing)
- Regression to earlier behaviors

**Golden Rule**: Only increase interval length when Remi is consistently calm at the current level.

## Practical Implementation Guide

### Before Each Session

1. **Assess readiness**: Is Remi calm before "get under"?
2. **Set interval target**: Know your start interval before entering
3. **Prepare treats**: Pre-count session treats to prevent overfeeding

### During the Session

1. **Use a silent cue** - A soft click or subtle hand signal instead of "good girl"
2. **Watch for settling** - Reward when Remi is already calm, not when she's restless
3. **Never extend if**: Barking, whining, pawing, or standing up
4. **Reset on breaks**: If Remi gets up, reset the interval clock

### Timing the Treats

**Option A: Phone Timer**
- Set timer for target interval
- When timer sounds, reward if calm
- Vary actual reward time by ±10-20 seconds

**Option B: Mental Tracking**
- Count seconds mentally
- Add randomness: "roughly 45 seconds... maybe 50"

## Troubleshooting

### If Remi Gets Frustrated (Pawing, Whining)

- **Step back**: Return to previous interval
- **Add more variation**: Unpredictability prevents frustration
- **Life rewards**: Mix in "release to sniff" as reward

### If Behavior Regresses

- Return to continuous reinforcement briefly (reward every calm moment)
- Reintroduce interval extension more slowly
- Check for external stressors (new environment, noise)

### Signs of Success

- Remi spontaneously settles without prompting
- Longer periods of eye contact/relaxation
- Less reliance on treats - responds to praise alone
- Can handle unexpected distractions

## Quick Reference Card

**Extending Intervals - Do's and Don'ts**

| DO | DON'T |
|----|-------|
| Start with consistent success | Rush to longer intervals |
| Vary timing (unpredictable to dog) | Use fixed, predictable timing |
| Reward calm, settled behavior | Reward restlessness |
| Use life rewards occasionally | Rely solely on food |
| Track data to measure progress | Train without documentation |
| Reset if Remi breaks settle | Continue ignoring protest behaviors |
