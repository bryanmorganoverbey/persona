### Recommended Spreadsheet Structure

If using a digital spreadsheet (Excel, Google Sheets), organize columns as follows:

```
Column Headers:
A: Session #
B: Date
C: Location Name
D: Location Type (Cafe/Restaurant/Event/Other)
E: Environment Familiarity (1=Familiar, 2=Somewhat, 3=New)
F: Start Time
G: End Time
H: Total Session Duration (minutes)
I: Calm Behavior Duration (minutes)
J: Number of Treats Given
K: Average Treat Interval (minutes)
L: Final Treat Interval (minutes)
M: Barking Occurred (Y/N)
N: Whining Occurred (Y/N)
O: Left Position (Y/N)
P: Crowd Level (1-5)
Q: Noise Level (1-5)
R: Distraction Type
S: Success Rating (1-10)
T: Notes
```

---

### Key Metrics to Track

**Primary Metrics:**
1. **Calm Duration** - Total minutes Remi remains settled
2. **Treat Interval Progression** - Increase in time between treats
3. **Success Rate** - Sessions meeting target duration / total sessions

**Secondary Metrics:**
4. **Distraction Recovery Time** - How quickly Remi returns to calm after disturbance
5. **Environment Novelty Score** - Success rate in new vs familiar locations
6. **Treatment Frequency** - Number of treats per session over time

---

### Progress Analysis Framework

**Weekly Review:**
- Average calm duration per session
- Average treat interval
- Success rate by location type
- Notes on patterns (time of day, crowd levels)

**Monthly Analysis:**
- Trend line for calm duration
- Treat interval progression curve
- Success rate in new environments
- Identify strongest/weakest conditions
