# Remi Public Behavior Training — Session Data Tracker

## Overview

This document provides a structured system to track duration of calm behavior per training session. The data will help measure Remi's progress and determine when to extend treat intervals.

---

## Session Log Template

### Required Data Points Per Session

| Field | Description |
|-------|-------------|
| **Date** | Session date (YYYY-MM-DD) |
| **Location** | Cafe/restaurant name |
| **Environment Type** | Familiar / New / Unfamiliar |
| **Start Time** | When Remi went under |
| **End Time** | When session ended |
| **Total Duration** | Minutes of calm behavior |
| **Treats Given** | Number of treats delivered |
| **Treat Interval** | Average time between treats (minutes) |
| **Incidents** | Barking/noise events (0, 1, 2+, etc.) |
| **Rating** | 1-5 scale (1=struggled, 5=perfect) |
| **Notes** | Observations, distractions, weather, etc. |

## Metrics Framework

### Key Performance Indicators

1. **Average Calm Duration** — Mean minutes per session
2. **Success Rate** — Sessions with 0 incidents / total sessions
3. **Treat Efficiency** — Duration per treat given
4. **Environment Progression** — New locations attempted
5. **Incident Frequency** — Incidents per session

### Progress Milestones

| Level | Duration | Treat Interval | Environment |
|-------|----------|----------------|--------------|
| Beginner | 10-15 min | 2-3 min | Familiar only |
| Intermediate | 20-30 min | 5-7 min | Familiar + 1 new |
| Advanced | 40-60 min | 10-15 min | Any environment |
| Master | 60+ min | 15-20 min | Extended outings |

---

## How to Use This Tracker

### During Each Session

1. Note start time when giving "get under" command
2. Record each treat time on a phone/notepad
3. Note any incidents (barking, movement, noise)
4. Record end time when leaving

### After Each Session

1. Transfer data to this log
2. Calculate total duration = End Time - Start Time
3. Calculate average treat interval = Total Duration / Treats Given
4. Note progress toward next milestone

### Monthly Analysis

- Calculate average session duration
- Note improvement from previous month
- Determine if ready to extend treat intervals
- Identify environments needing more practice
