## Treat Interval Progression Schedule

Based on variable-ratio reinforcement principles, the goal is to gradually reduce treat frequency while maintaining calm behavior.

### Phase 1: Baseline (Week 1)
- **Interval:** Every 30 seconds
- **Goal:** Confirm reliable "get under" response

### Phase 2: Initial Fading (Week 2)
- **Interval:** Every 60 seconds
- **Criteria to advance:** 3 consecutive sessions with ≤1 incident

### Phase 3: Intermediate Fading (Week 3)
- **Interval:** Every 90 seconds
- **Criteria to advance:** 3 consecutive sessions with 0 incidents

### Phase 4: Extended Fading (Week 4)
- **Interval:** Every 120 seconds (2 minutes)
- **Criteria to advance:** 3 consecutive sessions with 0 incidents

### Important Notes on Fading
- Do NOT skip phases — gradual reduction maintains behavior reliability
- If incidents increase, return to previous interval for 2+ sessions
- Use variable timing (not exactly on schedule) to build resilience
- Always reinforce calmness immediately if dog settles after being restless

## New Environment Exposure Protocol

### Before Going to New Location
- [ ] Scout the location (dog-friendly? quiet hours?)
- [ ] Identify exit points and quiet corners
- [ ] Bring high-value treats and a familiar mat/bed if used

### At New Location
1. Allow 2-3 minutes to observe and acclimate
2. Give "get under" command in a calm area
3. Start with familiar (shorter) treat intervals
4. If dog shows stress (pacing, heavy panting, whale eye), end session early
5. End on a positive note — even if short

### New Location Difficulty Rating
| Rating | Environment Type |
|--------|-----------------|
| 1 | Quiet familiar cafe, low foot traffic |
| 2 | Familiar cafe during busy hour |
| 3 | New cafe, quiet time |
| 4 | New cafe, moderate traffic |
| 5 | High-energy environment (live music, crowded) |

Target: Achieve Rating 3+ reliability before moving to Rating 5.
