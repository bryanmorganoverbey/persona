## Training Philosophy

Based on research from professional dog trainers, the key principles for Remi's training are:

1. **Positive Reinforcement** — Reward calm, quiet behavior with treats
2. **Progressive Difficulty** — Start with familiar cafes, then advance to new environments
3. **Set Up for Success** — Choose quiet times, appropriate table locations
4. **Consistency** — Same command ("get under"), same expectations every time

---

## Pre-Session Checklist

Before each cafe training session, ensure:

- [ ] Remi has had adequate exercise (walk/play) beforehand
- [ ] Remi has eliminated before arriving
- [ ] High-value treats are packed (more tempting than regular kibble)
- [ ] Water bowl available
- [ ] Leash and collar/harness ready
- [ ] Training mat or familiar blanket (optional, for comfort)

---

## Session Protocol

### Phase 1: Arrival & Setup (2-3 minutes)
1. Arrive during a quiet time if possible
2. Choose a table with adequate space beneath for Remi
3. Prefer spots with some shade and buffer from heavy foot traffic
4. Allow Remi to settle briefly before giving commands

### Phase 2: "Get Under" Command
1. Give clear command: **"Get under"**
2. Point to space under table/chair
3. Reward immediately when Remi is in position
4. Initial treat: within 5 seconds of compliance

### Phase 3: Treat Interval Schedule

| Training Stage | Treat Interval | Session Duration | Notes |
|----------------|----------------|------------------|-------|
| **Stage 1: Learning** | Every 30 seconds | 10-15 min | High frequency rewards for compliance |
| **Stage 2: Building** | Every 60 seconds | 15-20 min | Start extending slightly |
| **Stage 3: Strengthening** | Every 90 seconds | 20-30 min | Only if behavior is reliable |
| **Stage 4: Maintenance** | Every 2-3 minutes | 30+ min | Reduce frequency while maintaining |

### Phase 4: Handling Distractions
- **Other dogs passing by**: Keep Remi focused with treats if calm
- **People approaching**: Remi should remain under table; reward quiet
- **Food delivery**: Waiter brings food = high distraction; have treats ready
- **Barking/triggers**: If Remi reacts, redirect with treats; if continues, end session

### Phase 5: Session End
- Release with calm command: **"Okay"** or **"Free"**
- Praise enthusiastically when Remi exits calmly
- Do NOT reward if Remi bolts out excitedly

## Troubleshooting Common Issues

### Issue: Remi Won't Settle Under Table
- Practice "place" command at home first
- Bring familiar-smelling item (blanket, toy)
- Ensure adequate pre-exercise

### Issue: Barking at Other Dogs/People
- Move to quieter location or leave
- Do NOT reward attention-seeking barking
- Reward only quiet behavior
- Consider "hush" command training separately

### Issue: Gets Up Constantly
- Keep leash on loosely for guidance
- Return to shorter treat intervals
- Practice stay/release commands at home

### Issue: Begging for Food
- Never feed from table
- Use "leave it" command
- Keep treats in your hand, not on plate
