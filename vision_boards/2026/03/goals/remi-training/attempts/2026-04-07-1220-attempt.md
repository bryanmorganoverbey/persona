## Session Data Log Template

### Required Fields Per Session

| Field | Description |
|-------|-------------|
| **Date** | YYYY-MM-DD |
| **Location** | Name/type of venue (e.g., "Café Central", "New restaurant - Mexican") |
| **Environment Type** | Familiar / Unfamiliar / Challenging |
| **Session Start Time** | When "get under" command given |
| **Session End Time** | When session ended or Remi released |
| **Total Duration (minutes)** | Calculated duration of calm behavior |
| **Treats Given** | Number of treats during session |
| **Treat Interval (avg)** | Average time between treats in minutes |
| **Barking/Noise Events** | Count of any vocalizations (0 = perfect) |
| **Distractions Present** | List (e.g., "waiter passing", "other dog", "loud music") |
| **Remi's State** | Calm / Alert / Anxious / Success |
| **Notes** | Any observations on behavior quality |

---

## Progress Metrics to Track

### 1. Duration Metrics
- **Total session duration** - How long Remi maintained calm behavior under the table
- **Longest calm period** - Sustained calm without treats needed
- **Average session duration** - Rolling average across sessions

### 2. Treat Interval Metrics
- **Initial treat interval** - Time to first treat after "get under"
- **Extended interval** - Time between treats as training progresses
- **Treats per minute** - Decreasing number indicates progress

### 3. Behavioral Metrics
- **Barking incidents per session** - Target: 0
- **Noise alerts** - Any sounds/whining
- **Self-settling** - Did Remi settle without treats?

## Treatment Extension Progression Guide

Based on VCA Animal Hospitals guidance on calm training:

| Phase | Treat Interval | Session Duration Goal | Indicators of Success |
|-------|----------------|----------------------|----------------------|
| Phase 1 | Every 30-60 seconds | 5-10 minutes | Remi settles quickly |
| Phase 2 | Every 60-90 seconds | 10-15 minutes | No barking, relaxed posture |
| Phase 3 | Every 90-120 seconds | 15-20 minutes | Remi self-settles |
| Phase 4 | Every 2-3 minutes | 20-30 minutes | Reliable in familiar venues |
| Phase 5 | Variable/rare | 30+ minutes | Calm in unfamiliar environments |
