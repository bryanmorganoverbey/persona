## Suggested Workflow

### Before Each Session
1. Note the target treat interval (based on previous session's average)
2. Set a timer for tracking intervals during session
3. Record start time

### During Session
1. Log each treat given with timestamp
2. Note any incidents or breaks in calm behavior
3. Track when Remi settles vs. when disruptions occur

### After Session (Within 1 Hour)
1. Complete full session log
2. Calculate totals and averages
3. Fill in behavioral observations
4. Save to the CSV file

### Weekly
1. Review week's data
2. Calculate averages and trends
3. Adjust next week's treat interval targets
