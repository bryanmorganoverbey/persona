### Telegram Bot Setup Guide

The Autonomous Goal Execution System uses Telegram to send:
1. **Clarification questions** — when the agent is blocked and needs user input
2. **Run summaries** — after each 2-hour execution cycle

#### Step 1: Create Your Telegram Bot

1. Open Telegram and search for **@BotFather** (https://t.me/BotFather)
2. Send the command: `/newbot`
3. Follow the prompts:
   - **Name:** Choose a name (e.g., "Goal Agent" or "Vision Bot")
   - **Username:** Must end in `bot` (e.g., `my_goal_agent_bot`)
4. **BotFather will reply with your token** — it looks like:
   ```
   4839574812:AAFD39kkdpWt3ywyRZergyOLMaJhac60qc
   ```
5. **Save this token securely** — you'll need it for GitHub secrets

**Optional:** Customize your bot:
- `/setdescription` — add a description
- `/setabouttext` — add short bio
- `/setuserpic` — add a profile picture

---

#### Step 2: Get Your Telegram Chat ID

You need your personal chat ID to receive messages. There are several ways:

**Method A: Using @userinfobot (Easiest)**
1. Search for **@userinfobot** (https://t.me/userinfobot) on Telegram
2. Start the bot — send `/start`
3. The bot will reply with your user info, including your **chat_id**
   - Example: `Your user ID is: 123456789`

**Method B: Using @RawDataBot**
1. Search for **@RawDataBot** (https://t.me/RawDataBot)
2. Send `/start`
3. Look for `"id":123456789` in the response

**Method C: Using getUpdates API**
1. Open this URL in your browser (replace `YOUR_BOT_TOKEN`):
   ```
   https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates
   ```
2. Send a message to your bot first (start it with `/start`)
3. Refresh the URL — you should see JSON with `"id":123456789`
4. **Note:** This only works if no bot processes have read the updates

---

#### Step 3: Configure GitHub Secrets

1. Go to your GitHub repository
2. Navigate to **Settings** → **Secrets and variables** → **Actions**
3. Click **New repository secret** for each:

| Secret Name | Value | Example |
|------------|-------|---------|
| `TELEGRAM_BOT_TOKEN` | Your bot token from BotFather | `4839574812:AAFD39kkdpWt3ywyRZergyOLMaJhac60qc` |
| `TELEGRAM_CHAT_ID` | Your chat ID (number only) | `123456789` |
