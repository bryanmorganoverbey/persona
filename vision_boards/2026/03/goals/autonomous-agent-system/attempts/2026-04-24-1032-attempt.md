### Step 1: Create a New Telegram Bot

1. **Open Telegram** and search for **@BotFather** (look for the blue checkmark)
2. Start a chat with BotFather
3. Send the command `/newbot`
4. Follow the prompts:
   - **Name**: Enter a display name (e.g., "Goal Agent" or "Vision Board Bot")
   - **Username**: Enter a unique username ending in `_bot` (e.g., `MyGoalAgent_bot`)
5. **Save the bot token** — you'll get something like:
   ```
   4839574812:AAFD39kkdpWt3ywyRZergyOLMaJhac60qc
   ```
6. (Optional) Customize your bot:
   - `/setdescription` — add a description
   - `/setuserpic` — add a profile picture
   - `/setcommands` — add these commands:
     ```
     status - Check current goal status
     help - Get help
     pause - Pause the agent
     resume - Resume the agent
     ```

---

### Step 2: Get Your Chat ID

**Option A: Using @userinfobot (Easiest)**

1. Open Telegram and search for **@userinfobot**
2. Start a chat and send `/start`
3. The bot will reply with your user information, including your **Chat ID** (a positive number like `123456789`)

**Option B: Using the API Directly**

1. First, send a message to your new bot (open the bot and type anything)
2. Visit this URL (replace `<YOUR_BOT_TOKEN>`):
   ```
   https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
   ```
3. Look for `"id":123456789` in the `"chat"` object — that's your chat ID
4. If you don't see it, try sending a message to the bot first, then refresh

**Option C: Using a Gist (Automated)**

1. Use this online tool: https://clickalgo.com/telegram-chatid
2. Enter your bot token
3. It will display your chat ID

---

### Step 3: Add Secrets to GitHub

1. Go to your GitHub repository
2. Navigate to **Settings** → **Secrets and variables** → **Actions**
3. Click **New repository secret** and add:

| Secret Name | Value | Description |
|------------|-------|-------------|
| `TELEGRAM_BOT_TOKEN` | `4839574812:AAFD39kkdpWt3ywyRZergyOLMaJhac60qc` | Your bot token from Step 1 |
| `TELEGRAM_CHAT_ID` | `123456789` | Your chat ID from Step 2 |

### Troubleshooting

| Issue | Solution |
|-------|----------|
| Bot not responding | Make sure you've started a chat with the bot first |
| Chat ID not showing | Send a message to the bot, then check getUpdates |
| Messages not delivering | Verify token and chat ID are correct in GitHub secrets |
| Rate limiting | Telegram has limits; the agent is designed to send minimal messages |

---

### Security Notes

- **Never share your bot token** — it's like a password
- **Keep secrets in GitHub** — never hardcode in source files
- The bot only sends messages **to your chat** — it cannot receive commands autonomously (that would require a running service)
