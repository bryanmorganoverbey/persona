## 1. Create a Telegram Bot

### Step 1.1: Start BotFather

1. Open Telegram and search for **@BotFather**
2. Click Start to begin

### Step 1.2: Create the Bot

```
/newbot
```

Follow the prompts:
- **Bot name**: Enter a display name (e.g., "Goal Agent")
- **Username**: Must end in `_bot` (e.g., `MyGoalAgent_bot`)

### Step 1.3: Get the Bot Token

BotFather will respond with a message like:

```
Done! Congratulations on your new bot. You will find it at t.me/MyGoalAgent_bot.

Use this token to access the HTTP API:
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
Keep your token secure and store it safely, it can be used by anyone to control your bot.
```

**Save this token** — it will be your `TELEGRAM_BOT_TOKEN` secret.

---

## 2. Get Your Chat ID

### Method A: Using @userinfobot (Recommended for Personal Chat)

1. Search for **@userinfobot** on Telegram
2. Click Start
3. The bot will display your User ID (a number like `123456789`)

### Method B: Using web.telegram.org

1. Open https://web.telegram.org/
2. Log in to your account
3. Look at the URL in your browser address bar
4. The format is: `https://web.telegram.org/a/#/pvt?p=CHAT_ID`
5. The number after `p=` is your chat ID

### Method C: Using @RawDataBot

1. Search for **@RawDataBot** on Telegram
2. Start the bot
3. Forward any message to it
4. The bot will reply with raw data including your chat ID

### Method D: Using getUpdates API

> ⚠️ **Note**: As of 2025, this method may return empty results due to Telegram privacy mode changes. If you encounter this, use Method A, B, or C.

1. Open this URL in your browser (replace `BOT_TOKEN`):
   ```
   https://api.telegram.org/botBOT_TOKEN/getUpdates
   ```
2. You'll see JSON response with your chat ID in `result[].message.chat.id`

## 5. Telegram Integration Code

The project already has a `telegram.py` module at `scripts/goal-agent/telegram.py`. Below is the complete implementation:

```python
"""
Telegram integration for the Goal Agent.

Provides functions to send notifications and clarification questions.
"""

import os
import logging
import httpx
from typing import Optional

logger = logging.getLogger(__name__)


class TelegramClient:
    """Client for sending messages via Telegram Bot API."""
    
    API_URL = "https://api.telegram.org/bot{bot_token}/{method}"
    
    def __init__(self, bot_token: Optional[str] = None, chat_id: Optional[str] = None):
        """Initialize with Bot API token and target chat ID.
        
        Args:
            bot_token: Telegram Bot API token (from @BotFather)
            chat_id: Target chat ID to send messages to
        """
        self.bot_token = bot_token or os.environ.get("TELEGRAM_BOT_TOKEN")
        self.chat_id = chat_id or os.environ.get("TELEGRAM_CHAT_ID")
        
        if not self.bot_token:
            raise ValueError("TELEGRAM_BOT_TOKEN is required")
        if not self.chat_id:
            raise ValueError("TELEGRAM_CHAT_ID is required")
    
    def _request(self, method: str, data: dict) -> dict:
        """Make a request to the Telegram Bot API.
        
        Args:
            method: API method name (e.g., 'sendMessage')
            data: Request data dictionary
            
        Returns:
            API response as dictionary
            
        Raises:
            httpx.HTTPStatusError: If the API returns an error
        """
        url = self.API_URL.format(bot_token=self.bot_token, method=method)
        
        with httpx.Client(timeout=30.0) as client:
            response = client.post(url, json=data)
            response.raise_for_status()
            result = response.json()
            
            if not result.get("ok"):
                error_desc = result.get("description", "Unknown error")
                raise RuntimeError(f"Telegram API error: {error_desc}")
            
            return result.get("result", {})
    
    def send_message(
        self,
        text: str,
        parse_mode: str = "Markdown",
        disable_notification: bool = True,
        reply_markup: Optional[dict] = None
    ) -> dict:
        """Send a message to the configured chat.
        
        Args:
            text: Message text (supports Markdown formatting)
            parse_mode: Parse mode ('Markdown' or 'HTML')
            disable_notification: Send silently without notification sound
            reply_markup: Optional inline keyboard markup (JSON dict)
            
        Returns:
            Sent message object from API
        """
        data = {
            "chat_id": self.chat_id,
            "text": text,
            "parse_mode": parse_mode,
            "disable_notification": disable_notification,
        }
        
        if reply_markup:
            data["reply_markup"] = reply_markup
        
        logger.info(f"Sending Telegram message: {text[:100]}...")
        return self._request("sendMessage", data)
    
    def send_run_summary(
        self,
        tasks_attempted: int,
        tasks_completed: int,
        tasks_blocked: int,
        duration_minutes: float,
        details: Optional[str] = None
    ) -> dict:
        """Send a run summary notification.
        
        Args:
            tasks_attempted: Number of tasks attempted
            tasks_completed: Number of tasks completed
            tasks_blocked: Number of tasks blocked (needs clarification)
            duration_minutes: How long the run took
            details: Optional additional details
            
        Returns:
            Sent message object from API
        """
        emoji_status = "✅" if tasks_blocked == 0 else "⚠️"
        
        summary = f"""*{emoji_status} Goal Agent Run Summary*

• Attempted: {tasks_attempted}
• Completed: {tasks_completed}
• Blocked: {tasks_blocked}
• Duration: {duration_minutes:.1f} min"""

        if details:
            summary += f"\n\n{details}"
        
        return self.send_message(summary)
    
    def ask_clarification(
        self,
        goal_name: str,
        task_name: str,
        question: str,
        task_path: str
    ) -> dict:
        """Ask the user a clarification question for a task.
        
        Creates an inline keyboard with buttons to help the user respond.
        
        Args:
            goal_name: Name of the goal
            task_name: Name of the task
            question: The clarification question
            task_path: Path to the task file
            
        Returns:
            Sent message object from API
        """
        message = f"""*Clarification Needed* 🔍

**Goal:** {goal_name}
**Task:** {task_name}

{question}

_Please reply with your answer._
"""
        
        # Inline keyboard with quick response options
        reply_markup = {
            "inline_keyboard": [
                [
                    {"text": "Skip this task", "callback_data": f"skip:{task_path}"},
                    {"text": "Try again later", "callback_data": f"later:{task_path}"}
                ]
            ]
        }
        
        return self.send_message(message, reply_markup=reply_markup)
    
    def send_error(self, error_message: str) -> dict:
        """Send an error notification.
        
        Args:
            error_message: Error description
            
        Returns:
            Sent message object from API
        """
        message = f"""*Goal Agent Error* ❌

```
{error_message}
```

Please check the logs and fix the issue."""
        
        return self.send_message(message)


# --- Convenience functions ---

def get_client() -> TelegramClient:
    """Create a TelegramClient from environment variables.
    
    Returns:
        Configured TelegramClient instance
        
    Raises:
        ValueError: If required environment variables are missing
    """
    return TelegramClient()


def send_notification(text: str) -> dict:
    """Send a quick notification message.
    
    Args:
        text: Message text to send
        
    Returns:
        Sent message object from API
    """
    client = get_client()
    return client.send_message(text)


if __name__ == "__main__":
    # Test the bot when run directly
    import sys
    
    logging.basicConfig(level=logging.INFO)
    
    print("Testing Telegram bot connection...")
    
    try:
        client = get_client()
        result = client.send_message("🤖 Goal Agent is online! Testing complete.")
        print(f"Message sent successfully: {result['message_id']}")
        sys.exit(0)
    except Exception as e:
        print(f"Error: {e}")
        sys.exit(1)
```

## 8. Troubleshooting

| Problem | Solution |
|---------|----------|
| `getUpdates` returns empty | Disable privacy mode via @BotFather, or use @userinfobot to get chat ID |
| "Chat not found" error | Make sure you've started a chat with your bot first |
| "Bot blocked" error | The user has blocked the bot; they need to unblock it |
| Messages not arriving | Check that the bot has permission to message users |
| 403 Forbidden | Bot wasn't added to the group first (if sending to groups) |
