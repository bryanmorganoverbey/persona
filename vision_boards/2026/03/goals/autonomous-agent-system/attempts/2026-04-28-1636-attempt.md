# Telegram Bot Setup for Autonomous Goal Execution System

## Overview

The Telegram bot serves two purposes in the goal execution system:
1. **Run summaries** — After each GitHub Actions run, the agent sends a summary of tasks attempted
2. **Clarification questions** — When blocked, the agent can ask the user specific questions to proceed

---

## Step 1: Create Your Bot with BotFather

### Process

| Step | Action |
|------|--------|
| 1 | Open Telegram and search for `@BotFather` |
| 2 | Send the command `/newbot` |
| 3 | Choose a **display name** (e.g., "Goal Agent" or "Nick's Goal Bot") |
| 4 | Choose a unique **username** ending in `_bot` (e.g., `GoalAgentBot` or `nicksGoalBot`) |
| 5 | Receive your **bot token** — it looks like `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11` |

### Important Notes

- **Save the token immediately** — BotFather only shows it once
- The token is what you'll add as `TELEGRAM_BOT_TOKEN` in GitHub secrets
- You can always get a new token with `/revoke` if the old one is compromised

---

## Step 2: Get Your Chat ID

Your chat ID is your personal identifier on Telegram. You'll need it for two reasons:
1. To receive messages from your bot
2. To add as `TELEGRAM_CHAT_ID` in GitHub secrets

### Method 1: Using @userinfobot (Recommended)

| Step | Action |
|------|--------|
| 1 | Open Telegram and search for `@userinfobot` |
| 2 | Send the command `/start` |
| 3 | The bot will reply with your **User ID** — this is your chat ID |

### Method 2: Using @getidsbot

| Step | Action |
|------|--------|
| 1 | Open Telegram and search for `@getidsbot` |
| 2 | Send the command `/start` |
| 3 | The bot will display your chat ID in the response |

### Method 3: Manual (if bots don't respond)

| Step | Action |
|------|--------|
| 1 | Add your bot to a group (or use it in a direct message) |
| 2 | Visit: `https://api.telegram.org/bot<TOKEN>/getUpdates` |
| 3 | Look for `"chat":{"id":` in the JSON response |

## Step 5: Telegram.py Implementation

The complete module that handles messaging:

```python
"""
scripts/goal-agent/telegram.py

Telegram messaging for the Autonomous Goal Execution System.
Handles:
- Run summaries after each execution
- Clarification questions when blocked
"""

import os
import requests
from datetime import datetime
from typing import Optional

TELEGRAM_API_URL = "https://api.telegram.org/bot"


def get_bot_token() -> str:
    """Get bot token from environment or secrets."""
    token = os.environ.get("TELEGRAM_BOT_TOKEN")
    if not token:
        raise ValueError("TELEGRAM_BOT_TOKEN not configured")
    return token


def get_chat_id() -> str:
    """Get chat ID from environment or secrets."""
    chat_id = os.environ.get("TELEGRAM_CHAT_ID")
    if not chat_id:
        raise ValueError("TELEGRAM_CHAT_ID not configured")
    return chat_id


def send_message(
    text: str,
    parse_mode: str = "Markdown",
    disable_notification: bool = False
) -> bool:
    """
    Send a message via Telegram bot.
    
    Args:
        text: Message text (supports Markdown markdown)
        parse_mode: "Markdown" or "HTML"
        disable_notification: Send silently
    
    Returns:
        True if successful, False otherwise
    """
    token = get_bot_token()
    chat_id = get_chat_id()
    
    url = f"{TELEGRAM_API_URL}{token}/sendMessage"
    payload = {
        "chat_id": chat_id,
        "text": text,
        "parse_mode": parse_mode,
        "disable_notification": disable_notification
    }
    
    try:
        response = requests.post(url, json=payload, timeout=30)
        response.raise_for_status()
        result = response.json()
        return result.get("ok", False)
    except Exception as e:
        print(f"Failed to send Telegram message: {e}")
        return False


def send_run_summary(
    tasks_attempted: int,
    tasks_completed: int,
    tasks_blocked: int,
    blocked_tasks: list[dict],
    run_id: str
) -> bool:
    """
    Send a run summary after goal agent execution.
    
    Args:
        tasks_attempted: Total tasks worked on
        tasks_completed: Successfully completed
        tasks_blocked: Blocked requiring clarification
        blocked_tasks: List of {"task": str, "question": str}
        run_id: Timestamp or ID for the run
    
    Returns:
        True if successful
    """
    emoji_map = {
        "completed": "✅",
        "blocked": "⏳",
        "failed": "❌"
    }
    
    lines = [
        "🤖 *Goal Agent Run Complete*",
        "",
        f"📋 *Tasks Attempted:* {tasks_attempted}",
        f"{emoji_map['completed']} *Completed:* {tasks_completed}",
        f"{emoji_map['blocked']} *Blocked:* {tasks_blocked}",
        ""
    ]
    
    if blocked_tasks:
        lines.append("📌 *Blocked Tasks:*")
        for bt in blocked_tasks:
            lines.append(f"• {bt['task']}")
            lines.append(f"  → Question: {bt['question']}")
            lines.append("")
    
    lines.append(f"_*Run ID:* {run_id}")
    
    text = "\n".join(lines)
    return send_message(text)


def ask_clarification(
    task_name: str,
    task_context: str,
    question: str
) -> bool:
    """
    Ask a clarification question when blocked.
    
    Args:
        task_name: Name of the task
        task_context: Context/details about the task
        question: Specific question to ask
    
    Returns:
        True if message sent successfully
    """
    lines = [
        "⚠️ *Clarification Needed*",
        "",
        f"*Task:* {task_name}",
        "",
        f"_Context:_",
        f"{task_context}",
        "",
        f"I'm trying to complete this task but need some information:",
        "",
        f"❓ *{question}*",
        "",
        "Reply with your answer, and I'll continue the task.",
    ]
    
    text = "\n".join(lines)
    return send_message(text)


def test_connection() -> bool:
    """Test the bot connection."""
    return send_message(
        "🔔 *Goal Agent Connected!* ✅\n\n"
        "Your Telegram bot is now configured.\n"
        "I'll send run summaries and ask clarification questions here.",
        disable_notification=False
    )
```

## Step 7: Security Best Practices

| Practice | Why |
|----------|-----|
| Use GitHub secrets | Never commit tokens to code |
| Rotate tokens periodically | Limits exposure if compromised |
| Restrict bot permissions | Only grant needed permissions via BotFather |
| Don't share your chat ID publicly | Prevents spam from unknown bots |
| Use `/setprivacy` with BotFather | Restricts what commands the bot can see |

### BotFather Commands for Security

| Command | Purpose |
|---------|---------|
| `/setprivacy` | Enable to only receive commands you're explicitly mentioned in |
| `/setjoingroups` | Disable if you don't need group functionality |
| `/deletebot` | Remove a compromised bot |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "Chat not found" | Verify `TELEGRAM_CHAT_ID` is correct; start a chat with your bot first |
| "Bot blocked" | Unblock the bot in Telegram settings |
| "Token invalid" | Use `/token` with BotFather to get a new token |
| Messages not arriving | Check bot has permission to send messages; verify chat_id |
| "Too many requests" | You're hitting rate limits; wait and retry |
